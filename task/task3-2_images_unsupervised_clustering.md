# Учебное задание 3.2: Обучение без учителя — кластеризация медицинских изображений с использованием K‑Means

## Задача

> **У вас есть датасет** медицинских RGB‑изображений PathMNIST, содержащий образцы тканей, относящиеся к 9 различным патологиям. Каждое изображение имеет метку класса, но **вы не должны использовать эти метки при обучении** — это *обучение без учителя*.

> **Вы должны написать код для кластеризации медицинских изображений** в значимые группы, выполнив следующие шаги:
> 1. Загрузить и предобработать датасет PathMNIST.
> 2. Применить **K‑Means** для кластеризации в N кластеров (число соответствует истинному количеству классов).
> 3. Визуализировать результаты с помощью **t‑SNE** (или PCA + t‑SNE), центроидов кластеров, матрицы путаницы, примеров изображений из каждого кластера и распределения кластеров по истинным классам.
> 4. Оценить качество кластеризации с использованием **метрик**: Adjusted Rand Index (ARI), Normalized Mutual Information (NMI), Silhouette Score.

**Ожидаемый результат**: решение для кластеризации, которое группирует медицинские изображения в кластеры *без использования меток патологий в процессе обучения*, а также анализ качества полученных групп.

---

## Описание датасета

| Свойство | Значение |
|----------|----------|
| **Название** | PathMNIST (часть коллекции MedMNIST) |
| **Формат** | Массив NumPy (из библиотеки `medmnist`) |
| **Образцы** | 7180 изображений (тестовая выборка) |
| **Размер изображения** | 28×28 пикселей, 3 канала (RGB) |
| **Классы** | 9 (различные патологии тканей) |
| **Метки** | Целочисленные идентификаторы классов (только для оценки) |
| **Источник** | `medmnist.PathMNIST` |

## Требования к коду

✅ **Код** (`.ipynb` или `.py`), содержащий:
- Загрузку датасета и преобразование изображений в плоский вектор
- Реализацию K‑Means (можно использовать `sklearn.cluster.KMeans`)
- Расчёт метрик качества: **ARI**, **NMI**, **Silhouette Score** (на подвыборке, если время выполнения велико)
- Визуализации:
  - **t‑SNE** (или PCA + t‑SNE) проекция, раскрашенная по истинным меткам и по предсказанным кластерам
  - **Центроиды кластеров** (отображение усреднённых изображений для каждого кластера)
  - **Матрица путаницы** (Confusion Matrix) между истинными классами и кластерами
  - **Примеры изображений** из каждого кластера (по 5 случайных образцов)
  - **Столбчатая диаграмма** состава кластеров (процентное соотношение истинных классов в каждом кластере)

✅ **Краткий отчёт** (не более 2000 знаков):
- Какие методы кластеризации и предобработки использованы?
- Как Вы интерпретируете полученные метрики (ARI, NMI, Silhouette)?
- Какие визуализации помогают понять качество кластеризации и почему?

✅ **Визуализация в коде** должна быть сохранена в файлы (например, `1_tsne_comparison.png`, `2_centroids.png` и т.д.).
