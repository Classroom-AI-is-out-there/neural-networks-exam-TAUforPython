# Учебное задание 2.1: Обучение с учителем - классификация сердечных сокращений по ЭКГ

## Задача

> **У вас есть датасет** сегментов ЭКГ фиксированной длины, где каждый сегмент соответствует одному сердечному сокращению. Для каждого сегмента известен тип сердечного ритма, поэтому **метки нужно использовать при обучении** - это *задача обучения с учителем*.

> **Вы должны написать код для классификации сердечных сокращений**, выполнив следующие шаги:
> 1. Загрузить и предобработать данные из файла `dataset/task2_1-ecg5000-heartbeat-classification.csv`
> 2. Разделить данные на train / validation / test с сохранением баланса классов
> 3. Реализовать и обучить модель для одномерных временных рядов, например `1D CNN`, `GRU` или гибридную архитектуру
> 4. Оценить качество модели на тестовой выборке с помощью `Accuracy` и `macro F1-score`

**Ожидаемый результат**: рабочий код с комментариями, где реализована модель, классифицирующая сердечные сокращения по морфологии ЭКГ и объясняющая, какие типы сокращений распознаются лучше и хуже.

## Описание датасета

Датасет основан на наборе `ECG5000`, который часто используется для учебных задач по анализу временных рядов ЭКГ.
https://www.timeseriesclassification.com/description.php?Dataset=ECG5000 

| Свойство | Значение |
|----------|-------|
| **Формат** | CSV (`.csv`) |
| **Образцы** | 5000 сегментов ЭКГ |
| **Длина сигнала** | 140 временных точек |
| **Метки** | 5 классов сердечных сокращений |
| **Файл** | `task2_1-ecg5000-heartbeat-classification.csv` |

Поля в датасете:
- `source_split` - исходное разбиение датасета (`train` или `test`)
- `label_id` - числовой идентификатор класса
- `label_name` - название класса
- `t_000 ... t_139` - значения сигнала ЭКГ

Классы сердечного ритма:
- `N` Normal (Стандартный синусовый удар сердца: нормальная последовательность)
- `PVC` Premature Ventricular Contraction Внеочередное сокращение, возникающее в желудочках до ожидаемого синусового импульса.
- `R-on-T PVC`  R-on-T Premature Ventricular Contraction Особо опасная форма PVC, при которой преждевременный комплекс QRS накладывается на зубец T предыдущего сокращения
- `SP or EB` Supra-ventricular Premature or Ectopic beat Внеочередной импульс, комплекс QRS обычно узкий
- `UB` unclassified beat

Пример строки данных:
```text
train,1,N,-0.112,-2.827,-3.773,-4.350,-4.119,...
```
