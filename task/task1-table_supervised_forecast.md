# Учебное задание 1: Обучение с учителем — прогнозирование сердечно-сосудистого риска и преддиабета по данным диспансеризации

## Задача

У вас есть датасет результатов диспансеризации 1000 пациентов возрастом 40–65 лет. Для каждого пациента известны антропометрические, лабораторные и поведенческие показатели, а также целевые переменные: риск сердечно-сосудистых заболеваний (ССЗ) и статус глюкозы. Это задача **обучения с учителем** для медицинской классификации.

Вы должны написать код для решения двух подзадач:
1. **Прогнозирование высокого ССЗ-риска** (бинарная классификация)
2. **Оценка риска прогрессии в диабет** (нечёткая классификация для целевой группы)

Выполните следующие шаги:

### Шаг 1: Загрузка и предобработка данных
- Загрузить данные из файла `dataset/task_1_dispensarization_data_2026.csv`
- Обработать пропущенные значения в лабораторных показателях
- Выполнить стандартизацию признаков при необходимости

### Шаг 2: Разделение данных
- Разделить данные на train/validation/test с сохранением баланса классов (`stratify`)
- Для задачи преддиабета выделить целевую группу: пациенты с глюкозой 5.4–6.5 ммоль/л

### Шаг 3: Реализация и обучение моделей
- **Задача 1 (ССЗ-риск)**: Обучить модель классификации (SVM, Random Forest или логистическая регрессия) с балансировкой классов
- **Задача 2 (преддиабет)**: Реализовать нечёткий KNN-классификатор, возвращающий степени принадлежности к классам вместо жёсткой метки

### Шаг 4: Оценка качества
- Для ССЗ-риска: Accuracy, Recall, Specificity, Precision, F1-score, ROC-AUC
- Для преддиабета: анализ распределения скоров риска, метрики неопределённости прогнозов
- Построить матрицу ошибок и интерпретировать результаты

### Шаг 5: Клиническая интерпретация
- Объяснить, какие пациенты классифицируются правильно/ошибочно
- Проанализировать «серую зону» прогнозов и предложить тактику наблюдения
- Сформулировать рекомендации для врача на основе скоринга риска

## Описание датасета

Датасет содержит симулированные, но клинически реалистичные данные диспансеризации, сгенерированные с учётом медицинских корреляций между показателями.

| Свойство | Значение |
|----------|----------|
| Формат | CSV (UTF-8 с BOM) |
| Пациенты | 1000 человек |
| Возраст | 40–65 лет |
| Признаки | 15 переменных |
| Целевые переменные | 3: `ССЗ_риск_высокий`, `Статус_глюкозы`, `Доклинический_риск` |
| Файл | `dataset/task_1_dispensarization_data_2026.csv` |

### Поля в датасете:

| Поле | Тип | Описание |
|------|-----|----------|
| `Возраст` | int | Возраст пациента, лет |
| `Пол_мужской` | 0/1 | 1 = мужской, 0 = женский |
| `ИМТ` | float | Индекс массы тела, кг/м² |
| `Окружность_талии_см` | float | Окружность талии, см |
| `САД_мм_рт_ст` | float | Систолическое АД, мм рт. ст. |
| `ДАД_мм_рт_ст` | float | Диастолическое АД, мм рт. ст. |
| `Пульсовое_давление` | float | Разница САД и ДАД |
| `Глюкоза_натощак_ммоль_л` | float | Глюкоза плазмы натощак |
| `HbA1c_%` | float | Гликированный гемоглобин |
| `ЛПНП_ммоль_л` | float | Липопротеины низкой плотности |
| `ЛПВП_ммоль_л` | float | Липопротеины высокой плотности |
| `Триглицериды_ммоль_л` | float | Уровень триглицеридов |
| `СКФ_мл_мин` | float | Скорость клубочковой фильтрации |
| `Курение` | 0/1 | Статус курения |
| `Физическая_активность_мин_нед` | float | Физическая активность, мин/неделю |
| `ССЗ_риск_высокий` | 0/1 | Целевая: высокий сердечно-сосудистый риск |
| `Статус_глюкозы` | 0/1/2 | Целевая: 0=норма, 1=преддиабет, 2=диабет |
| `Доклинический_риск` | 0/1 | Целевая: комбинация факторов без диагноза |

## Подсказки для выполнения

1. Начните с разведочного анализа: постройте гистограммы признаков, оцените корреляции

5. Не забудьте про этические аспекты: модель — инструмент поддержки, а не замена врача
